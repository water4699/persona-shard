# Persona Shard Frontend

This is the React frontend for the Persona Shard mental health survey DApp.

## Environment Variables

Create a `.env` file in this directory with the following variables:

```env
# WalletConnect Project ID (get from https://cloud.walletconnect.com/)
VITE_WALLETCONNECT_PROJECT_ID=your_walletconnect_project_id

# Sepolia RPC URL for testing (optional, uses public RPC by default)
VITE_SEPOLIA_RPC_URL=https://sepolia.infura.io/v3/YOUR_INFURA_PROJECT_ID
```

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
